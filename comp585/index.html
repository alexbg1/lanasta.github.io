Client: Carolyn Quinsey (Carolyn.Quinsey@unchealth.unc.edu)
Organization: UNC Neurosurgery
Topic: Neurosurgery Resident Education
Objective: Radiology and Pathology flashcard like learning
Audience: Neurosurgery residents
Background: Neurosurgery residents need mental breaks when studying to retain information. Often they feel guilty when doing intense study to take the necessary breaks for retention. The game would provide enough humor and mental downtime so that information can be retained and meaningful.

Current Stage

Initial setup for game development. The game will be web based. We will be using Github as source control and hoping to utilize Phaser.io as the game framework. The languages we will be using are Javascript, HTML5 and CSS3.

Meetings with Client

1. Wednesday, 2/3/2016

  • We were given a clearer description of how the client envisions the game to be
  • Our goal is to cater to the neurosurgery residents' humor which is mostly immature
  • Client wants a game leaderboard and a user database
  • Client wants a web based game due to residents having different models of mobile phones
